## üìå Project Highlights

* **Dataset:** Utilized the `spam.csv` collection for training and testing.
* **High Accuracy:** Achieved **98.2% accuracy** using Support Vector Machines (SVM).
* **NLP Pipeline:** Automated text cleaning, tokenization, and stop-word removal.
* **Feature Extraction:** Implemented **TF-IDF Vectorization** to convert text into numerical vectors.
